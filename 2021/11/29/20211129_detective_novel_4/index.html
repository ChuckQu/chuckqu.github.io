狂读推理小说的一年（四）

发表于 2021-11-29 更新于 2022-05-08 分类于 读书

米泽穗信《寻狗事务所》

  1. 用两个第一人称视角分别跟踪两个案件，在读者的视角下这两个案件显然是密切关联的，但两个‘我’一开始并不知道，这种视角上的落差带来悬念感。

  2. 米泽穗信擅长于第一人称文本的心理活动描写，尤其是半平这个角色，设定是一个头脑简单，做着侦探梦的中二青年形象，他的心理活动描写也就比较‘粗犷’，和米泽一贯的那种‘细腻缜密’的心理描写不同。

  3. 故事核心是一个‘猎杀和反猎杀’的故事，“桐子”这个角色贯穿始终，但直到最后才露面，故事的逆转也是她的形象的逆转，再次证明米泽笔下没有软弱的女性角色。而那个真正的反派‘变态骚扰者’，够阴险，不够狡诈，是一大遗憾。

  4. 推理的两个破局点一个很现代一个很古代。‘现代’的是通过追踪网络痕迹发现真相的过程（小说创作时还是博客时代，读起来令人怀念）。

  5. ‘古代’的部分通过追踪一个古文书的背景，揭开‘小伏’的历史，因为只有找到某本历史文献才能定位‘谷中城’的位置，‘桐子’借此设局，这就巧妙地把历史和现实的案件联系了起来，这点上有点像《追想五断章》了。

  6. 案件真相揭示的过程和‘我’的个人经历形成了呼应。案件中的桐子回到乡下，一开始看起来像是被动躲避，真相却是在布局主动猎杀。‘我’从东京回到乡下开办S&R，也是有逃避的意味的，经历这一案件后也许也不一样了。

有栖川有栖《月光游戏》

  1. 作者在后记里写了自己的这篇长篇处女作面世的曲折经历，他自己心比天高，一心只想冲江户川乱步奖，但坦率来讲，这本处女作尽管一改再改，但水平确实不高。
  2. 故事是经典的‘暴雪山庄’模式，但出场人物还是太多了，如果要写一个17个人的密室案件，挑战的其实是作者的塑造人物的能力，这方面做得并不好，一直到结局，看到人名还要去扉页查询一下。
  3. 真相和动机最后还是落在少男少女的爱情萌芽和脆弱的心理。可以理解年轻人冲动之下连续杀人，但杀人时又那么冷静又缜密，是有点不能说服人的。
  4. 中间穿插了一些干扰项，比如神神叨叨的‘月人派’的女孩，几个角色回忆自己的童年经历，按套路来说这些看似‘闲笔’的地方会埋伏一点线索，到最后发现是纯粹的闲笔，这大概就是主角是‘推理社团’的反套路了。
  5. 挑战读者的模式失败的，因为塑造人物不成功，让读者去回想细节，一个一个人物进行推断和排除，是不现实的。

森博嗣《全部成为F》

  1. 文字比较干巴，读起来不流畅，作者试图追求一种‘理科式’的简洁冷酷的美感，但简洁和流畅是不矛盾的，可能是作者笔力不逮，后来看了豆瓣评论更可能是翻译的原因。
  2. 隔离的小岛和研究所，封闭的房间，是典型的密室推理。
  3. 主角少女的形象并不讨喜。
  4. 一众配角形象也很单调，全员工具人，不过倒是和程序员的身份很匹配。
  5. 题目的‘全部成为F’是最重要的一个诡计， 这种诡计真的是只有码农才想得出了，但有些人可能体会不到其中妙处。
  6. 密室如何凭空多出一个人？ 被长期关在里面，秘密怀孕生子并抚养长大，而外界毫无所知，这也是一个神奇的构想。
  7. 根源是伦理悲剧。
  8. 动机到最后也没解释清楚，犯人远超常人的智商并不能消解其行为上的不合情理。

乙一《箱庭图书馆》

  1. 又是一本短篇故事集，源于一个改写别人小说的企划，挺有意思。

  2. 和刚看过的《寂寞的频率》一样，仍保持了各种风格的混搭，青春、恐怖、悬疑、治愈、黑色幽默等各种标签都全了。

  3. 大部分主角也是典型的乙一人物，学校里的边缘人物，不合群，孤独感，爱写作，爱幻想。

  4. 《白色足迹》是标签式的奇幻温情路线，乙一在后记中承认一开始就觉得故事和自己以往的风格太像，主要是为了满足写一回推理小说的雪地足印的套路，才决定改编。

  5. 《奇境》写一个少年捡到一把钥匙，然后到处乱逛去试门，结果撞破变态杀人犯的恐怖故事。这个创意在星新一的短篇里也看到过。

东野圭吾《从前我死去的家》

  1. 东野圭吾认为是自己最被低估的小说，看封底编辑部也认为这是东野被低估的佳作。
  2. 是那种大部分人会一口气读完的小说，东野的流畅文笔，悬念的设置，略有点恐怖的氛围，加上篇幅不长，拿起就停不下来。
  3. 这篇最特别的地方是极简设定，两个人，一栋楼，两天内发掘一个家庭被掩埋的故事，推理小说写复杂是容易的，在有限的设定下编故事更显功底。
  4. 绝佳创意是建一栋和原楼一样的新楼当作坟墓，想来仍觉发冷。
  5. 本质还是社会派，反思父母和孩子的关系，关注儿童健康成长。

歌野晶午《想你，在樱树长满绿叶的季节》

  • 看豆瓣评论，非常有争议的叙述性诡计，很多推理迷被‘骗’得很不爽。

  • 如果没有这一诡计，看上去是一个正常的多线叙事的侦探故事，侦探成濑将虎接受委托步步接近真相，犯人古屋节子步步走入犯罪的深渊，并把下一个杀人目标定为侦探的朋友安藤士郎，两条线即将汇合。

  • 中间穿插了成濑将虎的爱情线和回忆线。

  • 推理小说读者在面对多线叙事时，马上就会怀疑背后的叙述性诡计，比如，回忆线里的黑帮故事和蓬莱俱乐部有什么联系。爱情线里的宫本樱和这个案件有什么联系。

  • 如果没有人物年龄这个核心叙诡，读者是很容易把古屋节子和宫本樱联系起来的。

  • 但是因为这个诡计，读者脑海中古屋节子是一个老太太，宫本樱是青春少女，谁都不会把两人划上等号。

  • 作者给读者设套从第一页就开始了，成濑将虎旺盛的性欲，日常去健身馆，在读高中生的朋友，爱玩的妹妹，是典型的年轻人的形象。

  • 而男女主角日常对话的节奏和语言也完全好似年轻情侣，难怪有读者说看到最后再回头看会觉得不舒服。

  • 这是不是也在挑战读者心中的成见呢？叙述性诡计有趣的地方就在于破解我们的思维定势。

  • 作者不止欺骗读者，成濑将虎还玩了一把李代桃僵，把女主角也被骗了，这样男女主角两个人都是在以假身份谈情说爱，故事更有趣了。

  • 大量着笔于揭露蓬莱俱乐部如何欺骗老人，也是这篇小说社会派的一面。

  • 最后回忆线的案子和主线关系不大，单独拎出来倒是一个规规矩矩的推理短篇，一个监守自盗的故事。

黑田研二《婚纱》

  • 小说一上来就是双线叙事，并且营造出一种某一时刻平行宇宙分叉的错觉，摆明了说“我在玩叙述性诡计”。

  • 诡计的复杂之处在于人物“身份”切换了多次，一层是叙述视角的‘欺骗’，作者在人物和时间线上都耍了点花招，另一层是故事里人物在玩‘欺骗’。

  • 小说的节奏非常紧凑，所有人好像都在急冲冲地赶路。两条线的故事进展都太快了，有时要回头去看一些细节。

  • 回头来看，这是一个非常复杂非常大的案子，但所有人都是在一个很小的地方兜兜转转，这种反差给人一种看舞台剧的感觉。

藤原伊织《恐怖分子的洋伞》

  • 读的第二本藤原伊织的小说，但已经大致可以总结出他的风格和套路了。
  • ‘硬汉派’推理多少和武侠小说有点像，藤原的小说尤其有‘古龙’的味道。
  • 男主角身患酒精中毒症， 总是威士忌不离身，年轻时打拳击，练就一身硬本领。在破案的过程中，总是混在自己的‘丐帮’朋友中寻找线索， 和浅井是酷酷的男人间的友情，到最后发现旧时老友才是大反派，这是典型的古龙小说的主角和故事线。
  • 和塔子的感情线，最后看起来有点像倚天的杨不悔的故事。
  • 回忆线中的背景是全共斗学生运动，现实线是江口组黑社会，加了点社会派的味道。
  • 和《离别的火焰》一样，有一段关键的美国往事。

西泽保彦《死了七次的男人》

  • 推理小说和‘土拨鼠之日’式的科幻设定的结合。
  • 西泽保彦的风格是很轻松愉快的，这从每章的标题就可以看出来。每隔一定篇幅还会很‘贴心’地帮读者总结一下，理一下思路。
  • 角色有点动漫化的夸张，比如里面的人物经常互相吐槽，某些时候的思维和行为不够贴近‘真实’，看上去就会觉得是‘漏洞’。
  • 两个核心诡计很厉害，一个是把循环开始的日期搞错，另一个是漏掉一个循环，尤其是后一个，看到‘你在某一个循环里死了’的时候，真是有点头皮发麻。
  • 但是有一些情节是有点牵强的。比如外公对两个女婿的陷害，外公在多个循环里被同样的方法杀死的解释。

柳广司《代号D机关Ⅰ》

  • 以间谍为背景的斗智式的短篇小说集。
  • 最大特点就是层层反转，每个故事都是‘螳螂捕蝉黄雀在后’。
  • 节奏是非常平的，明明应该是非常紧张刺激的故事，作者用一种‘不动声色’的间谍视角，非常冷静地讲述出来。
  • 看到这个系列有很多部，暂时不往下看了。

山田正纪《诱饵搜查官之东京地铁杀人魔》

  • 非常老派，天然的电视剧剧本。

  • 主线大概就是一个女性搜查官通过破案成长的过程，搭配一个有责任感有个人魅力的上司，一个表面迷糊实际老练的搭档，穿插一点本厅和片区刑警的冲突情节，是一个‘样板式’的日式刑侦剧。

  • 包括高潮部分的反转也是‘样板式’的电视剧桥段。

  • 女警做诱饵，故事多少带了点情色味道。

  • 一行一句话的形式非常古龙了。

  • 最后这个犯罪动机也是挺匪夷所思的，丈夫负责地铁上猥亵女性，妻子负责杀害被猥亵的女性，奇怪的‘搭档’。

  • 虽然评论说山田正纪是日本‘国民级’文学大师，至少这本书看不出来，文字感觉挺粗糙的。

秋吉理香子《相亲中毒》

  • 围绕‘相亲’这个主题的四个小故事。
  • 两个怪人：一个变态女中介，一个不达目的誓不罢休的理科女。
  • 两组骗子：一个美女和丑女的搭档，通过制造感官差把同一个男人吃两遍；一个‘骗子’家庭通过相亲来骗彩礼。
  • 比较轻松好读。

森村诚一《人性的证明》

  1. 社会派的经典作品。
  2. 多线叙事，一开始看上去没有联系，到后面汇合起来组成一个完整的画卷，这也成了后来社会派的一种经典叙事模式。

凑佳苗《往复书简》

  1. 三则书信体的小故事，总体来说是‘充满爱意’的，逐步揭示真相的过程其实都是一个‘救赎’的过程。
  2. 故事并不复杂，尤其是前两个故事。但书信体的有意思的地方就在于，通过不同的人的交流拼接处一个完整的真相，而每个身处故事里的人却只看到自己认为的真相，有点‘罗生门’的意思。
  3. 第三则故事反转最多，也是最‘有爱’的一篇。

米泽穗信《十米真相》

  1. 主角是一个通过‘自由撰稿’赚钱的记者，所以除了通过推理发现案件真相，也有很多对于记者这个身份的探讨。总体而言，太刀洗万智这个角色是塑造的成功的。

  2. 几个小故事频繁切换叙事视角，这是米泽穗信一向爱玩的。《十米真相》是侦探（太刀洗）视角，《正义汉》是犯人视角，后面几篇类似搭档or路人视角。

  3. 相比后面几篇我反而更欣赏《正义汉》这个简单直接的推理。发现死的位置是中段车厢判断是他杀（自杀都选车头，速度大），在人流高峰时刻杀人判断是激情杀人，而不是精密计划过的，注意到死者死前有不文明行为，猜测很大可能是确信犯杀人；以上完成杀人犯身份推理，之后基于‘这种正义汉会留在犯罪现场见证’的推断开始设坑。
