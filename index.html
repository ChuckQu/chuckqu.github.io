工作随想（一）

发表于 2021-07-29 分类于 工作

做大和做小

数据部门的一大工作内容是做dashboard,所谓monitor the business。之前的一个困扰是这样的，给A部门做了一个dashboard，B部门看了，说哎不错，给我们也做一个，然后C部门也过来说我们也需要一个。这样一来你同样的dashboard要重复开发多次，而这个开发过程中每次底表都需要基于业务从头设计，绝不是简单的复制粘贴，花费的人力和时间就成倍增加了。有了这样的教训后，以后开发dashboard前先拉个会，把所有业务方都叫过来，把你们的需求都说清楚。我宁愿一开始就把这个dashboard做大，也要避免未来重复性的劳动。

后来要做一个客户分群的项目，基于同样的想法，也把所有的业务方拉过来，谈谈你们希望这个分群结果出来是什么样，能够解决什么问题。然后我们就希望一个分群能够解决所有业务方的问题。结果费了九牛二虎之力，还是可耻地失败了，出来了一个四不像的东西。然后就反思，这个项目一开始就没必要做得这么大，不应该指望一揽子解决所有问题。其实业务方的每个具体的问题都有针对性的更优的解决办法。所以应该一开始就做小，把每一个问题拆成一个小项目去做，反而效果更好。

客户分群（Segmentation）

几乎每个数据分析师都做过分群项目，它简单易操作，同时因为是非监督的，又存在很大的操作和解释空间，可以说是half science half art。而对于很多甲方或者业务方来说，分群的思想和结果易于理解，每个人都可以插进来发表一点意见，所以一直广受欢迎。

我之前在咨询公司，参与过和围观过的分群项目也不少，但真正称得上成功的凤毛麟角。所有的分群项目，到最后都要面对一个灵魂拷问：你这个结果看上去很合理很漂亮，但它能拿来干嘛，我们可以基于它做什么action?

上面提到的失败的分群项目，失败的一大原因也在于我们过于追求‘action’,总指望分群结果能指导运营和marketing，而不仅仅是“拿来看看而已”。

吸取教训后，我现在的认知是，分群就是适合于面向领导的“拿来看看”类的项目。如果你希望用它来解决什么问题，那个问题一定有更好的解法。比如你希望用它来做基于segments的产品推荐，那建一个完全个性化的推荐算法是绝对更优的方案。

最近读的推理小说（2）

发表于 2021-07-21 更新于 2021-08-25

继续总结推理小说

郑执《生吞》

青春小说和破案小说的结合，推理成分不强。

在看的过程中有些部分是让人嫌弃的，那种过于浓烈的青春小说的伤感味道，年纪大了不大能接受。而另一条老冯探案的线呢，看上去也不够冷不够硬汉。

但看到最后仍会动容，能感受到作者的动情。

后来在B站看了郑执在一席的演讲，小说和他自己的生活经历密切相关，老冯这个角色也投射了很多他对自己父亲的情感。

青崎有吾《风之丘五十元硬币之谜》

之前看过青崎有吾的水族馆之谜，没看完。

这本也算是短篇小故事集，就好比是打补丁一样，从时间上来看，是穿插在青崎有吾之前几本大案件故事之间的小故事，讲故事之余，把人物塑造得更完整。

写校园生活的文字和对话是很可爱的，每一篇的“案件”和“真相”也是可爱的。乱扔的餐盘是因为女朋友做了不合口味的便当，五十元硬币是因为小孩子们想多捡点钱，乐队唯一的男生受“霸凌”是因为女孩子要脱衣服散热，阅读体验是轻松愉快的。

呼延云《扫鼠岭》

国产推理中口碑比较高的，看完比较失望。

最差的是文笔，突出的一点是不说人话，里面的人物经常脱口而出一大段带着很多排比和成语的慷慨陈词，现实中绝不会有人这样说话。中国的推理圈难以更上一个台阶，一大原因就是这些作者的基本文学素养太差，还处于爱好者自high的阶段。

推理小说首先是小说，，否则诡计再精巧，社会性再深刻，也是白搭。

然后是人物，设置了三个破案如神的主要人物形象，完全看不出有什么必要，然后这三个人的塑造又是非常强的动漫气息。其他像郭记者那一对也完全是工具人。为了表现自己是“社会派”推理，构建了一个邪恶的犯罪集团，但这个集团到底有多么强大，则处于一种非常薛定谔的状态。

核心推理诡计是“最好的谜面，是虚假的谜底”，凶手的设计是：首先铺设第一层非常明显的假线索，让警察马上抓到自己；然后抛出第二层假线索，让警察开始怀疑自己并非真凶，并逐步洗清嫌疑；真正的最后一层真线索当然需要主角侦探去发现了。

里面的很多推理看起来很牵强，立不住脚。

陈浩基《遗忘，刑警》

在京东读书读完《扫鼠岭》，看到下面的相关推荐有这本，打开很快就读完了。

第三次读陈浩基，不过这本是他早期的作品，所以相比后期的要稚嫩一点。

对于推理小说，精神分裂的设定有点像作弊，不过警察和嫌疑犯的双重精神分裂的设定倒很独特，形成了更复杂的层次结构。

文笔嘛，还是那种很干瘪，很程序员的风格。

绫辻行人 《钟表馆事件》

馆系列的经典，

东野圭吾 《放学后》

东野圭吾的出道成名作，用西上心太的话来说，正是这一部让东野被打上了“青春推理作家”的标签，

东野圭吾 《怪人们》

东野圭吾早期的短篇集，整体质量一般般，大概就是世界奇妙物语的水准。

短篇推理，更容易让人记住的是奇特的动机，而不是奇诡的推理。几篇中我最有印象：

《至死方休》： 兢兢业业的上班族，误以为自己是被失控的机器人用机械臂击中，担心这一事故曝出来对公司造成不良影响，于是迷糊着走到休息室，并制造密室，让自己死的更干净些，不想连累公司。

《灯塔之上》：‘我’在旅游景点差点遭遇性侵，逃出来后，“诱骗‘ 一同长大的好友去旅游景点，有一点黑暗系的味道了。

《蜜月之旅》：’我‘自己的失误害死了自己的孩子，妻子不想让我发现真相后产生负罪感，于是制造假象和作伪证，结果反惹起’我‘的疑心。

其他几篇更着重诡计和推理，倒没什么印象了。

最近读的推理小说

发表于 2021-06-25 更新于 2021-09-16

看推理小说总结诡计和套路

阿加莎·克里斯蒂《ABC谋杀案》

连环杀人推理领域的开创性作品，核心诡计是两个：

  1. 若干案件中只有一个是真案件，其他案件起“掩护”功能。真案件如果单独拉出来看，动机和最大嫌疑人非常明显，在放在一系列连环案件中，人们太专注于发现案件之间的联系，而忽略了单个案件的事实。
  2. 真凶找了一个‘代理人’，设计诱导这个人在不知情的情况下出现在各个案发现场，从而构成了一个明显的假线索，误导警察和侦探。

两个套路：

  1. 凶手一开始就向”著名“侦探波洛发起挑战来预告案件，侦探的存在成了凶手完成自己计划的关键一环。
  2. 推理小说中警察方一般都是用来反衬侦探的英明神武，这里经常和波洛一起讨论案件的是年轻有为的克鲁姆警官和一位心理学医生。每当讨论到案件中的疑点时，警官和医生负责通过心理学进行精神分析，把这个疑点圆过去。心理学大概一直是推理作家心目中的”反派“，推理的核心是发现不合逻辑之处，而心理学就厉害在可以解释一切。

东野圭吾《麒麟之翼》

这本小说太接近一个现成的电影剧本了，所以也有很多影视的套路。

  1. 侦探个人生活中面临的困境和案件真相是”主题统一“的，这里都是‘父子关系’困境。
  2. 案件表面搭配一点神秘主义的意象，比如这里的麒麟像下的死亡，死者祭拜七福神的行为。
  3. 一个表面上看起来非常”完美“的假嫌疑人，事实上他是”巧合“出现在案发现场，更”巧合“的是他和死者竟然是现实中可以联系起来的。
  4. 祭拜的目的从”求子“转向”溺水”是关键，这里有点生硬。
  5. 赎罪真是永恒的主题。

米泽穗信《满愿》

是短篇小说集，第一次看这个作家，之前知道他是以“青春推理”出名，看完才发现是走黑暗路线的推理小说。

好几篇的心理刻画让我想到乙一，搭配如《石榴》这种略显病态的主题，读完只觉寒意袭人。

最喜欢的是《夜警》，一开始就摆出谜题，然后回忆的部分用很多细节来丰满这个人物，而答案就埋藏在这些细节之中！

秋吉理香子《圣母》

非常精巧的叙述性诡计。

多视角叙事是叙诡的常见结构。其中的两个视角，看上去是完全不相干的人，但其实是生活在同一屋檐下的。这时候如何蒙蔽读者呢？作者设计了另一个小叙诡，很狡猾地掩藏了主角的性别，直到最后才揭示，但这一点很重要，是前面的诡计成立的关键。

这本书前面大部分给人感觉是冰凉不适的，到最后一下子呈现出一种母性光辉般的崇高感。

叶真中显《死亡护理师》

真是非常“硬”的社会派推理小说了，定下一个主题-日本的老龄化问题，一个切入点-老年护理行业，然后开始搭积木一样往里塞人物，代表“善”的检察官，代表“恶”的企业高管，绝望的护理行业底层从业人员，棱角分明，直来直去，中间大段议论文般在探讨日本“老无所依”的困境。

推理只是点缀的小把戏，但也有亮点。一是玩了一个小小的叙述性诡计，多线叙事，其实凶手视角和某个主角视角是重合的；结尾一个小小的反转，给读者的错觉是主角在捕猎“真凶”。另一个有意思的地方，后面破解案件的关键一步是靠着助理检察官基于Excel做的数据分析，虽然用到的统计方法很简单，大概就是基于数据分布发现outlier, 但也算是展现了“小数据”的威力。

从这本书还可以了解点一点日本的司法制度，摘抄一下：

日本的刑事司法号称“精密司法”，判决有罪率99.9%，检察官基于侦查结果做核实，只对确定有罪的案件起诉，所以日本的法庭不是争论有罪无罪的地方，而是执行父爱主义的裁决，检察官向被告问罪，辩护律师陈述被告如何悔过，法官进行教导促其改过。可以看到，日本的法庭审判极具宗教性和道德性。

日本的检察官制度，检察官有侦查权，但很少直接参与案件侦查，大部分工作是于办公室核实案件。

检察官称为“独任官员”，只有检察官有起诉权，哪怕警方逮捕，如果检察官不起诉，也只能无罪释放。

检察官是真正决定一个人有罪与否的人，也可以说是唯一具有合法杀人权力的人，考虑到死刑是最高刑罚。

检察官责任重大，不允许独断专行，要求无条件服从以检察长为首的指挥系统，每个案件都必须向上级汇报，获得批准。

最近读了很多东野圭吾

发表于 2021-06-16 更新于 2021-06-17 分类于 读书

我自己分析主要是两个原因：

一是今年以来工作压力变大，每天上班都感觉有干不完的活，这样下班的时候基本处于身心俱疲的状态。地铁上还是坚持着看书的习惯，但只适合读一些轻松点的休闲小说了。

二是养成了从图书馆借书读的习惯，而去图书馆大部分时候我是从新书区挑书看，结果经常发现怎么东野圭吾又有新书上架了，在纠结于借哪本的时候，东野圭吾就成了比较安全的选择。

也曾试过找比如阿加莎克里斯蒂的书，但家附近的图书馆还是差了一点，竟然找不到阿婆的书，福尔摩斯的书倒有一些。

因为轻松好读，半年以来读过的东野圭吾的书有近十本。真论水准，也是良莠不齐，不过这么高产的情况下有些写烂了也可以理解，阿加莎克里斯蒂也高产，作品中同样也有不少水准之下的。

除了量大更新快，东野圭吾的另一大特点是题材和风格的多样化。以最近看过的几本举例，黎明之街的恋爱心理描写胜过很多爱情小说，超杀人事件和黑笑小说之类专门“编排”推理圈，其中的幽默感也独树一帜，学生街的日子充满青春小说的颓丧感，祈年守护人是和解忧杂货店一样的温情鸡汤路线，我想这也是东野圭吾的一个厉害之处。 很多推理小说家都会有一个特定的标签，比如“本格”，“崩坏”，“暗黑”，同样的风格看几本就会腻。而东野圭吾的标签却很难总结，硬要套一个也只能是“高产”，“流行”，“通俗”之类了。如果碰到那种只能带一个推理作家的书去荒岛的问题，我的答案大概还是东野圭吾。

也正因为太通俗流行（现在进任何一个书店都可以看到至少一大排东野圭吾，文峰的新华书店在畅销区有一个专门的书架全是他的书），完全可以理解某些推理小说迷对他的嫌弃。之前听某播客讲，上海某高校的推理协会在招新时会问你最喜欢的小说家是谁，回答东野圭吾的就会拒掉。

今日的东野圭吾绝对算是功成名就，很多作家在这个地位都会变得爱惜自己的羽毛，惜墨如金。东野圭吾却是反过来越写越多，也许再难有匹及早期嫌疑人和白夜行之类的作品，但这种写作的热情也很令人佩服。

读书笔记《大汉之怒》

发表于 2021-05-23 更新于 2021-05-25 分类于 读书

这本书读完后印象最深的是封底的作者简介，84年生（比我还年轻），北大历史学博士，通晓英语法语德语，能阅读日文俄文阿拉伯文蒙古文等十多种文字。一眼看上去就很厉害。

书是很快能翻完的，一方面是因为是比较薄的“小史”，另一方面是因为作者文笔生动有趣，读起来很顺畅，这一点尤为可贵。中国的很多年轻历史学者，学术功底非常扎实，但不会写面向大众的历史通俗读物，写出的东西过于严肃或晦涩，对于普通读者不太友好，是非常可惜的。

这本书纠正了我以前的一个错误印象，就是书一开头引用鲁迅回忆留日学生的话：”那时倘非天幸，这岛国早被我们灭掉了！“。我以前也是认为：当时元朝军事实力对日本有碾压的优势，没有征服日本只是因为台风，老天爷作梗。

但读完这本书的感受是，就算没有台风，元朝也是不大可能征服日本的。

忽必烈征日的愿望是很强烈的，但元朝一直没有派出最强实力，两次出征的军队都只能算是杂牌军。东路军中虽然也有蒙古军队，但主要的还是高丽人，包括船只也全是高丽制造。第二次的江南军则全是南宋的降军，所谓的人类史上最大舰队，也几乎全是南宋的旧战船。蒙古人精的很，几乎就是驱使着高丽人和南宋人做炮灰。

在台风之前，双方是打了一些遭遇战的，从书里描写来看，难分胜负，没有一方有明显优势。而考虑到日本基本只是出动了九州的兵力，本州的援军还没有加入，可以猜想，就算没有台风，元军也很难往前进了。

但假设忽必烈抽调最强的蒙古军再打一次会怎样？还是很难说，因为跨海作战，加上日本的地理环境，还有情报工作的缺乏，蒙古军队的很多优势体现不出来，大概率还是陷入苦战。

这本书更有趣的写战争前的部分。尤其是高丽夹在中间的尴尬位置，他见识过蒙古人的凶悍，被蒙古人当作征日前线基地，也只能逆来顺受；他也知道当时日本武士之国的可怕，所以受蒙古人指使去给日本人投战书，还要小心地试图避免激怒日本。他也深知若两国真打起来，自己可能是最遭殃的，所以中间还搞些外交小动作试图止战。这让我想起之后甲午战争还有日俄战争时朝鲜其实都是最大的受害方，这大概就是朝鲜半岛这块土地一直以来的悲哀吧。
