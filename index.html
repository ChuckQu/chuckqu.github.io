东京女子图鉴

发表于 2022-05-07

这个电视剧如果当作一篇东京游记看也挺有意思的。

女主角绫初到东京，先去港区瞻仰心中圣地，号称“名媛罐头”的六本木之丘。
找房子一开始也是先去港区的南青山一带，然后发现这一带生活很不方便。带看的房产中介背后吐槽：因为住在这儿的人都有车，不需要去搭电车，也不用去便利店，对于穷人来说当然是不”方便“的。
绫当时月薪23万，中介按照三分之一收入的原则，还是推荐她去吉祥寺或下北泽，最后租在了差不多价位的三茶。

接下来绫游走于东京各个地标：

  • 三茶被形容为“有烟火气”的地方，适合新鲜人

  • 工作两三年就该搬到”时尚“一点的惠比寿了

  • 在银座变身为”白领精英“

  • 婚后在丰州这种”中产“聚集的地方买房

  • 离婚后如”贵妇“般 生活在代代木

港区被反复提及是东京最高端的地区，绫后面几任男友有两个港区男，一个目黑男。除了南青山和六本木，西麻布也被提及。没有提及的是白金，刚刚看《卖房子的女人》，里面有段对话：

三轩家赚这么多钱，应该住哪里呢？
应该住白金吧

地图上看，港区，目黑，涉谷这几个高端区是挨着的，港区的特点是有很多外国领事馆，涉谷则有松涛这种政治人物多的地方。

位于江东区的丰州被描绘成中产阶级聚集的地方，当时绫年薪大概800万，丈夫是亚马逊的中层，年薪900万，合起来年薪大概1700万。
查一下，日本30岁的平均年薪才400多万，所以这时已经不是普通的中产阶级了。

正因此，他们的房子在丰州也算是比较高端的，剧里重点提了房子是高层，这里有一个小细节表现了同一栋楼”高层“和”低层“的差距，一个住在低楼层的女孩在按电梯按钮时还刻意掩饰一下自己内心的自卑感。

最近看河濑直美的电影《晨曦将至》，里面有句台词：”你们家住三十层，应该不会这么小气“，是同样的意思。

东京房子有多贵？看另一部日剧《卖房子的女人》可以有一些粗浅的印象，一般一室一厅是两三千万日元，两室两厅是四五千万，对比上海看起来也不算贵。里面一个流浪汉社长买了一个可以看到富士山景的别墅，原价也不过一亿五千万日元（最后被三轩家卖出了三亿）。提到最贵的一个房子是南青山的，报价是七亿日元。

狂读推理小说的又一年（一）

发表于 2022-01-05 更新于 2022-01-06

牧萨次《完全恋爱》

  1. 与其说是本格推理，更像是一个画家悲剧一生的年代小说。
  2. 小说时间跨度非常大，从二战结束一直写道泡沫经济破灭，本庄究从少年到暮年，为了自己想象中的恋爱以一生守护一个女人的四代‘分身’。
  3. 第一个案件像是引子，后两个案件布局很有气势，但解密有点潦草，第二个‘地球最大密室’的谜底是少男少女的殉情，第三个‘不在场证明’的破解是连体双胞胎，尽管回头来看，作者做了不少铺垫，不算突兀，但总有点泄气。
  4. 与案件的破解相比，最后戳破‘完全恋爱’的这个反转才更令人唏嘘，从旁观者角度来看他是最不幸的，但于他自己呢？
  5. 更恰当的比喻是恋爱小说中插入了几个推理梗，作者特地让好几个角色变成推理小说迷，最后为了说清真相，不惜‘自己’亲自上阵。

方丈贵惠《时空旅行者的沙漏》

  1. 之前没法想象推理小说怎么和时空穿越的科幻小说相结合，因为时空穿越必然导致悖论，很可能就破环了推理的逻辑。
  2. 但作者很聪明，科幻设定主要用来圆‘动机’，犯罪手法和推理破局仍在传统推理小说的框架内。
  3. 杀人手法中也用到了‘穿越’的手段，但因为加了种种‘合理’的限制，所以是在‘可推理’的范围内的，并不突兀。
  4. 和‘科幻‘设定搭配的是非常老套的’暴雪山庄‘+’家族诅咒‘的推理老梗，也形成了有趣的反差。
  5. 推理部分的结构有点像《月光游戏》，作者和有栖川一样都是京都大学毕业。

狂读推理小说的一年（四）

发表于 2021-11-29 更新于 2021-12-29

米泽穗信《寻狗事务所》

  1. 用两个第一人称视角分别跟踪两个案件，在读者的视角下这两个案件显然是密切关联的，但两个‘我’一开始并不知道，这种视角上的落差带来悬念感。

  2. 米泽穗信擅长于第一人称文本的心理活动描写，尤其是半平这个角色，设定是一个头脑简单，做着侦探梦的中二青年形象，他的心理活动描写也就比较‘粗犷’，和米泽一贯的那种‘细腻缜密’的心理描写不同。

  3. 故事核心是一个‘猎杀和反猎杀’的故事，“桐子”这个角色贯穿始终，但直到最后才露面，故事的逆转也是她的形象的逆转，再次证明米泽笔下没有软弱的女性角色。而那个真正的反派‘变态骚扰者’，够阴险，不够狡诈，是一大遗憾。

  4. 推理的两个破局点一个很现代一个很古代。‘现代’的是通过追踪网络痕迹发现真相的过程（小说创作时还是博客时代，读起来令人怀念）。

  5. ‘古代’的部分通过追踪一个古文书的背景，揭开‘小伏’的历史，因为只有找到某本历史文献才能定位‘谷中城’的位置，‘桐子’借此设局，这就巧妙地把历史和现实的案件联系了起来，这点上有点像《追想五断章》了。

  6. 案件真相揭示的过程和‘我’的个人经历形成了呼应。案件中的桐子回到乡下，一开始看起来像是被动躲避，真相却是在布局主动猎杀。‘我’从东京回到乡下开办S&R，也是有逃避的意味的，经历这一案件后也许也不一样了。

有栖川有栖《月光游戏》

  1. 作者在后记里写了自己的这篇长篇处女作面世的曲折经历，他自己心比天高，一心只想冲江户川乱步奖，但坦率来讲，这本处女作尽管一改再改，但水平确实不高。
  2. 故事是经典的‘暴雪山庄’模式，但出场人物还是太多了，如果要写一个17个人的密室案件，挑战的其实是作者的塑造人物的能力，这方面做得并不好，一直到结局，看到人名还要去扉页查询一下。
  3. 真相和动机最后还是落在少男少女的爱情萌芽和脆弱的心理。可以理解年轻人冲动之下连续杀人，但杀人时又那么冷静又缜密，是有点不能说服人的。
  4. 中间穿插了一些干扰项，比如神神叨叨的‘月人派’的女孩，几个角色回忆自己的童年经历，按套路来说这些看似‘闲笔’的地方会埋伏一点线索，到最后发现是纯粹的闲笔，这大概就是主角是‘推理社团’的反套路了。
  5. 挑战读者的模式失败的，因为塑造人物不成功，让读者去回想细节，一个一个人物进行推断和排除，是不现实的。

森博嗣《全部成为F》

  1. 文字比较干巴，读起来不流畅，作者试图追求一种‘理科式’的简洁冷酷的美感，但简洁和流畅是不矛盾的，可能是作者笔力不逮，后来看了豆瓣评论更可能是翻译的原因。
  2. 隔离的小岛和研究所，封闭的房间，是典型的密室推理。
  3. 主角少女的形象并不讨喜。
  4. 一众配角形象也很单调，全员工具人，不过倒是和程序员的身份很匹配。
  5. 题目的‘全部成为F’是最重要的一个诡计， 这种诡计真的是只有码农才想得出了，但有些人可能体会不到其中妙处。
  6. 密室如何凭空多出一个人？ 被长期关在里面，秘密怀孕生子并抚养长大，而外界毫无所知，这也是一个神奇的构想。
  7. 根源是伦理悲剧。
  8. 动机到最后也没解释清楚，犯人远超常人的智商并不能消解其行为上的不合情理。

乙一《箱庭图书馆》

  1. 又是一本短篇故事集，源于一个改写别人小说的企划，挺有意思。

  2. 和刚看过的《寂寞的频率》一样，仍保持了各种风格的混搭，青春、恐怖、悬疑、治愈、黑色幽默等各种标签都全了。

  3. 大部分主角也是典型的乙一人物，学校里的边缘人物，不合群，孤独感，爱写作，爱幻想。

  4. 《白色足迹》是标签式的奇幻温情路线，乙一在后记中承认一开始就觉得故事和自己以往的风格太像，主要是为了满足写一回推理小说的雪地足印的套路，才决定改编。

  5. 《奇境》写一个少年捡到一把钥匙，然后到处乱逛去试门，结果撞破变态杀人犯的恐怖故事。这个创意在星新一的短篇里也看到过。

东野圭吾《从前我死去的家》

  1. 东野圭吾认为是自己最被低估的小说，看封底编辑部也认为这是东野被低估的佳作。
  2. 是那种大部分人会一口气读完的小说，东野的流畅文笔，悬念的设置，略有点恐怖的氛围，加上篇幅不长，拿起就停不下来。
  3. 这篇最特别的地方是极简设定，两个人，一栋楼，两天内发掘一个家庭被掩埋的故事，推理小说写复杂是容易的，在有限的设定下编故事更显功底。
  4. 绝佳创意是建一栋和原楼一样的新楼当作坟墓，想来仍觉发冷。
  5. 本质还是社会派，反思父母和孩子的关系，关注儿童健康成长。

歌野晶午《想你，在樱树长满绿叶的季节》

  • 看豆瓣评论，非常有争议的叙述性诡计，很多推理迷被‘骗’得很不爽。

  • 如果没有这一诡计，看上去是一个正常的多线叙事的侦探故事，侦探成濑将虎接受委托步步接近真相，犯人古屋节子步步走入犯罪的深渊，并把下一个杀人目标定为侦探的朋友安藤士郎，两条线即将汇合。

  • 中间穿插了成濑将虎的爱情线和回忆线。

  • 推理小说读者在面对多线叙事时，马上就会怀疑背后的叙述性诡计，比如，回忆线里的黑帮故事和蓬莱俱乐部有什么联系。爱情线里的宫本樱和这个案件有什么联系。

  • 如果没有人物年龄这个核心叙诡，读者是很容易把古屋节子和宫本樱联系起来的。

  • 但是因为这个诡计，读者脑海中古屋节子是一个老太太，宫本樱是青春少女，谁都不会把两人划上等号。

  • 作者给读者设套从第一页就开始了，成濑将虎旺盛的性欲，日常去健身馆，在读高中生的朋友，爱玩的妹妹，是典型的年轻人的形象。

  • 而男女主角日常对话的节奏和语言也完全好似年轻情侣，难怪有读者说看到最后再回头看会觉得不舒服。

  • 这是不是也在挑战读者心中的成见呢？叙述性诡计有趣的地方就在于破解我们的思维定势。

  • 作者不止欺骗读者，成濑将虎还玩了一把李代桃僵，把女主角也被骗了，这样男女主角两个人都是在以假身份谈情说爱，故事更有趣了。

  • 大量着笔于揭露蓬莱俱乐部如何欺骗老人，也是这篇小说社会派的一面。

  • 最后回忆线的案子和主线关系不大，单独拎出来倒是一个规规矩矩的推理短篇，一个监守自盗的故事。

黑田研二《婚纱》

  • 小说一上来就是双线叙事，并且营造出一种某一时刻平行宇宙分叉的错觉，摆明了说“我在玩叙述性诡计”。

  • 诡计的复杂之处在于人物“身份”切换了多次，一层是叙述视角的‘欺骗’，作者在人物和时间线上都耍了点花招，另一层是故事里人物在玩‘欺骗’。

  • 小说的节奏非常紧凑，所有人好像都在急冲冲地赶路。两条线的故事进展都太快了，有时要回头去看一些细节。

  • 回头来看，这是一个非常复杂非常大的案子，但所有人都是在一个很小的地方兜兜转转，这种反差给人一种看舞台剧的感觉。

藤原伊织《恐怖分子的洋伞》

  • 读的第二本藤原伊织的小说，但已经大致可以总结出他的风格和套路了。
  • ‘硬汉派’推理多少和武侠小说有点像，藤原的小说尤其有‘古龙’的味道。
  • 男主角身患酒精中毒症， 总是威士忌不离身，年轻时打拳击，练就一身硬本领。在破案的过程中，总是混在自己的‘丐帮’朋友中寻找线索， 和浅井是酷酷的男人间的友情，到最后发现旧时老友才是大反派，这是典型的古龙小说的主角和故事线。
  • 和塔子的感情线，最后看起来有点像倚天的杨不悔的故事。
  • 回忆线中的背景是全共斗学生运动，现实线是江口组黑社会，加了点社会派的味道。
  • 和《离别的火焰》一样，有一段关键的美国往事。

西泽保彦《死了七次的男人》

  • 推理小说和‘土拨鼠之日’式的科幻设定的结合。
  • 西泽保彦的风格是很轻松愉快的，这从每章的标题就可以看出来。每隔一定篇幅还会很‘贴心’地帮读者总结一下，理一下思路。
  • 角色有点动漫化的夸张，比如里面的人物经常互相吐槽，某些时候的思维和行为不够贴近‘真实’，看上去就会觉得是‘漏洞’。
  • 两个核心诡计很厉害，一个是把循环开始的日期搞错，另一个是漏掉一个循环，尤其是后一个，看到‘你在某一个循环里死了’的时候，真是有点头皮发麻。
  • 但是有一些情节是有点牵强的。比如外公对两个女婿的陷害，外公在多个循环里被同样的方法杀死的解释。

柳广司《代号D机关Ⅰ》

  • 以间谍为背景的斗智式的短篇小说集。
  • 最大特点就是层层反转，每个故事都是‘螳螂捕蝉黄雀在后’。
  • 节奏是非常平的，明明应该是非常紧张刺激的故事，作者用一种‘不动声色’的间谍视角，非常冷静地讲述出来。
  • 看到这个系列有很多部，暂时不往下看了。

山田正纪《诱饵搜查官之东京地铁杀人魔》

  • 非常老派，天然的电视剧剧本。

  • 主线大概就是一个女性搜查官通过破案成长的过程，搭配一个有责任感有个人魅力的上司，一个表面迷糊实际老练的搭档，穿插一点本厅和片区刑警的冲突情节，是一个‘样板式’的日式刑侦剧。

  • 包括高潮部分的反转也是‘样板式’的电视剧桥段。

  • 女警做诱饵，故事多少带了点情色味道。

  • 一行一句话的形式非常古龙了。

  • 最后这个犯罪动机也是挺匪夷所思的，丈夫负责地铁上猥亵女性，妻子负责杀害被猥亵的女性，奇怪的‘搭档’。

  • 虽然评论说山田正纪是日本‘国民级’文学大师，至少这本书看不出来，文字感觉挺粗糙的。

秋吉理香子《相亲中毒》

  • 围绕‘相亲’这个主题的四个小故事。
  • 两个怪人：一个变态女中介，一个不达目的誓不罢休的理科女。
  • 两组骗子：一个美女和丑女的搭档，通过制造感官差把同一个男人吃两遍；一个‘骗子’家庭通过相亲来骗彩礼。
  • 比较轻松好读。

森村诚一《人性的证明》

  1. 社会派的经典作品。
  2. 多线叙事，一开始看上去没有联系，到后面汇合起来组成一个完整的画卷，这也成了后来社会派的一种经典叙事模式。

凑佳苗《往复书简》

  1. 三则书信体的小故事，总体来说是‘充满爱意’的，逐步揭示真相的过程其实都是一个‘救赎’的过程。
  2. 故事并不复杂，尤其是前两个故事。但书信体的有意思的地方就在于，通过不同的人的交流拼接处一个完整的真相，而每个身处故事里的人却只看到自己认为的真相，有点‘罗生门’的意思。
  3. 第三则故事反转最多，也是最‘有爱’的一篇。

米泽穗信《十米真相》

  1. 主角是一个通过‘自由撰稿’赚钱的记者，所以除了通过推理发现案件真相，也有很多对于记者这个身份的探讨。总体而言，太刀洗万智这个角色是塑造的成功的。

  2. 几个小故事频繁切换叙事视角，这是米泽穗信一向爱玩的。《十米真相》是侦探（太刀洗）视角，《正义汉》是犯人视角，后面几篇类似搭档or路人视角。

  3. 相比后面几篇我反而更欣赏《正义汉》这个简单直接的推理。发现死的位置是中段车厢判断是他杀（自杀都选车头，速度大），在人流高峰时刻杀人判断是激情杀人，而不是精密计划过的，注意到死者死前有不文明行为，猜测很大可能是确信犯杀人；以上完成杀人犯身份推理，之后基于‘这种正义汉会留在犯罪现场见证’的推断开始设坑。

狂读推理小说的一年（三）

发表于 2021-09-15 更新于 2021-11-19

米泽穗信《追想五断章》

很喜欢，以前看三体看到第三部，觉得很精彩的一个部分就是云天明的童话。

叶真中显《blue》

之前看过叶真中显的《死亡护理师》，感觉就是非常“硬”的社会派。这本更是“硬”到家了，可以说是纪录片式的推理小说，通过blue这个人物串起了昭和平成令和三个年代，串起了各式各样的人物。如果做个词频统计，“泡沫”估计是这本书里出现次数最多的词了，泡沫经济从繁荣到破灭，日本社会出现无数裂痕，这本书写的就是坠入裂缝的这些人。

叶真中显毫不掩饰自己的野心，就是一本书写尽平成年代。真相揭示的过程非常平实，毫不花哨。反而大量着墨于展示各种经济和文化元素，以及各种“配角”。泡沫破灭后从皮条客转业为劳务派遣的人，在滨松汽车工厂打工的日裔巴西人，小团块世代，失落的一代，由盛而衰的多摩新城，所有这些元素拼成了平成年代。

米泽穗信《羔羊的盛宴》

一系列以女仆为主角的暗黑小说，所有的人物通过一个叫巴比伦会的神秘结构串联起来。

和《满愿》一样，最大的感受是米泽穗信的心理描写真是一绝，尤其是以第一人称视角写’变态‘心理，用最平静和日常的语言，逐渐堆砌出最可怕的真相。

藤原伊织《离别的火焰》

《忽左忽右》最近有一期邀请了吴非聊推理小说的话题，谈到最近看的小说时吴非提到了藤原伊织，于是去上海图书馆的时候特地去日本小说书架翻了一下，借到了这一本。

  1. 最近一直看各种‘本格’和‘社会派’，偶尔看这种‘冷硬派’的还不大习惯。

  2. ‘冷硬派’的关键是塑造一个让人印象深刻的主角。主角‘我’对坏人下手毫不留情，和警察针锋相对，但却有一帮情谊深厚的‘酒友’，在探案的过程中一点点揭示和死者柿岛的友情羁绊之深，是非常典型的‘外冷内热’。我的感觉是好像古龙小说里的人物。

  3. 揭示真相的叙事结果是很清晰的，一开始以为是商业竞争，最后还是情感纠纷。 所有的一切完全源于几个主要人物在美国的一段往事，还是会觉得动机和推理过程比较勉强。

  4. 破案过程中还穿插了一个严肃的老头董事长和奔放的酒吧女老板的爱情线，比较老派。

  5. 印象深刻的一个‘推理点’：作为‘帮凶’的丸山在回答时说和柿岛告别后转身就走了，不记得柿岛走了哪条路。‘我‘马上指出其中的破绽：在日本职场，所有人都有一个习惯，和身居高位的上司道别时，都会先目送对方的背影走开一段距离，自己才离开。—这真的是非常’日本‘了。

  6. 看豆瓣上别人的介绍，发现藤原伊织也真是一个经历非常传奇的人物了。

  7. 作者结合自己广告界的工作经验，描写了一点日本便利店的生存困境，倒也有一点’社会派‘的味道了。

东野圭吾《信》

  1. 怎么都不能算是推理小说。
  2. 可以称为“教化小说”，就是简单讲一个人的经历，好比是给读者一枚镜子照照自己，然后反复提醒：“你是怎么想的呢？”。
  3. 叙事没有任何花招和技巧，完全直线叙事，直贵遇见了什么困难，做出了怎样的应对。如白石和社长平野这样的角色，在普通推理小说中可能会是一个更‘深’的角色，但这里就是扮演类似‘天使’和‘神父’的角色。
  4. 我以为的东野最大的优点-文字的流畅感，在这种结构下更显功底，基本是一口气读完。
  5. 对于’歧视‘这个主题探讨得很深入了，我印象深刻的是’歧视‘和’反向歧视‘。
  6. 提到了’公园初登场‘这个日本生活中特有的重要的场合。

乙一《寂寞的频率》

看上去是四篇风格迥异的短篇，有青春的，荒诞的，恐怖的，但都有一点的伤感的味道。

包括《抓住小偷的手》这一篇，前面是荒诞好笑的，但最后两个人握手后回眸一笑，也带出一点’就此别过‘的既甜蜜又伤感的味道。

《失去的世界》这一篇是最厉害的，应该很少有小说会以植物人的第一视角来写，更别提另一个更有创意的点，’我‘只有手臂有知觉，妻子是一个钢琴师，然后通过在手臂弹钢琴进行沟通，真是绝了。

看过几本乙一，此君’开脑洞‘之才，非一般人可比。

东野圭吾《沉默的巡游》

  1. 作为伽利略的新一篇，内核可以说是《嫌疑人X的献身》相似：为保护爱的人愿意自我牺牲。
  2. 叙事结构的层次性非常清晰，第一层似乎是在探讨：如果法律和检察系统对于那些保持沉默的嫌疑犯没有办法，民众是不是可以自发进行复仇？
  3. 到这一步，构造了一个’接力杀人‘的场景，这种’接力杀人‘的构思，东野圭吾在《布鲁特斯的心脏》里也构思过。
  4. 真相变复杂变有趣的地方在于，杀人计划因为种种主观或客观的因素，每一个环节都发生了变化。这时候，已经没有一个人能够把握真相了。
  5. 唯有汤川学拼出了真相。
  6. 最后落脚的转折从主题来说其实是有点’轻‘的，但对汤川学这个系列来说，有解脱的意味。

叶真中显《绝叫》

  1. 连续看的第三本叶真中显，到这一本，才是真的服了。当得上是社会派推理的巅峰。
  2. 和《死亡护理师》和《blue》一样，延续了这种纪录片式的文字风格，保留了多线叙事的结构。
  3. 第二人称视角相比《blue》里’致blue’的那条线，让读者更接近地去去了解阳子这个人物。

工作随想（一）

发表于 2021-07-29 分类于 工作

做大和做小

数据部门的一大工作内容是做dashboard,所谓monitor the business。之前的一个困扰是这样的，给A部门做了一个dashboard，B部门看了，说哎不错，给我们也做一个，然后C部门也过来说我们也需要一个。这样一来你同样的dashboard要重复开发多次，而这个开发过程中每次底表都需要基于业务从头设计，绝不是简单的复制粘贴，花费的人力和时间就成倍增加了。有了这样的教训后，以后开发dashboard前先拉个会，把所有业务方都叫过来，把你们的需求都说清楚。我宁愿一开始就把这个dashboard做大，也要避免未来重复性的劳动。

后来要做一个客户分群的项目，基于同样的想法，也把所有的业务方拉过来，谈谈你们希望这个分群结果出来是什么样，能够解决什么问题。然后我们就希望一个分群能够解决所有业务方的问题。结果费了九牛二虎之力，还是可耻地失败了，出来了一个四不像的东西。然后就反思，这个项目一开始就没必要做得这么大，不应该指望一揽子解决所有问题。其实业务方的每个具体的问题都有针对性的更优的解决办法。所以应该一开始就做小，把每一个问题拆成一个小项目去做，反而效果更好。

客户分群（Segmentation）

几乎每个数据分析师都做过分群项目，它简单易操作，同时因为是非监督的，又存在很大的操作和解释空间，可以说是half science half art。而对于很多甲方或者业务方来说，分群的思想和结果易于理解，每个人都可以插进来发表一点意见，所以一直广受欢迎。

我之前在咨询公司，参与过和围观过的分群项目也不少，但真正称得上成功的凤毛麟角。所有的分群项目，到最后都要面对一个灵魂拷问：你这个结果看上去很合理很漂亮，但它能拿来干嘛，我们可以基于它做什么action?

上面提到的失败的分群项目，失败的一大原因也在于我们过于追求‘action’,总指望分群结果能指导运营和marketing，而不仅仅是“拿来看看而已”。

吸取教训后，我现在的认知是，分群就是适合于面向领导的“拿来看看”类的项目。如果你希望用它来解决什么问题，那个问题一定有更好的解法。比如你希望用它来做基于segments的产品推荐，那建一个完全个性化的推荐算法是绝对更优的方案。

呼延云《扫鼠岭》

国产推理中口碑比较高的，看完比较失望。

最差的是文笔，突出的一点是不说人话，里面的人物经常脱口而出一大段带着很多排比和成语的慷慨陈词，现实中绝不会有人这样说话。中国的推理圈难以更上一个台阶，一大原因就是这些作者的基本文学素养太差，还处于爱好者自high的阶段。

推理小说首先是小说，，否则诡计再精巧，社会性再深刻，也是白搭。

然后是人物，设置了三个破案如神的主要人物形象，完全看不出有什么必要，然后这三个人的塑造又是非常强的动漫气息。其他像郭记者那一对也完全是工具人。为了表现自己是“社会派”推理，构建了一个邪恶的犯罪集团，但这个集团到底有多么强大，则处于一种非常薛定谔的状态。

核心推理诡计是“最好的谜面，是虚假的谜底”，凶手的设计是：首先铺设第一层非常明显的假线索，让警察马上抓到自己；然后抛出第二层假线索，让警察开始怀疑自己并非真凶，并逐步洗清嫌疑；真正的最后一层真线索当然需要主角侦探去发现了。

里面的很多推理看起来很牵强，立不住脚。

狂读推理小说的一年（一）

发表于 2021-06-25 更新于 2021-11-17

看推理小说总结诡计和套路

阿加莎·克里斯蒂《ABC谋杀案》

连环杀人推理领域的开创性作品，核心诡计是两个：

  1. 若干案件中只有一个是真案件，其他案件起“掩护”功能。真案件如果单独拉出来看，动机和最大嫌疑人非常明显，在放在一系列连环案件中，人们太专注于发现案件之间的联系，而忽略了单个案件的事实。
  2. 真凶找了一个‘代理人’，设计诱导这个人在不知情的情况下出现在各个案发现场，从而构成了一个明显的假线索，误导警察和侦探。

两个套路：

  1. 凶手一开始就向”著名“侦探波洛发起挑战来预告案件，侦探的存在成了凶手完成自己计划的关键一环。
  2. 推理小说中警察方一般都是用来反衬侦探的英明神武，这里经常和波洛一起讨论案件的是年轻有为的克鲁姆警官和一位心理学医生。每当讨论到案件中的疑点时，警官和医生负责通过心理学进行精神分析，把这个疑点圆过去。心理学大概一直是推理作家心目中的”反派“，推理的核心是发现不合逻辑之处，而心理学就厉害在可以解释一切。

东野圭吾《麒麟之翼》

这本小说太接近一个现成的电影剧本了，所以也有很多影视的套路。

  1. 侦探个人生活中面临的困境和案件真相是”主题统一“的，这里都是‘父子关系’困境。
  2. 案件表面搭配一点神秘主义的意象，比如这里的麒麟像下的死亡，死者祭拜七福神的行为。
  3. 一个表面上看起来非常”完美“的假嫌疑人，事实上他是”巧合“出现在案发现场，更”巧合“的是他和死者竟然是现实中可以联系起来的。
  4. 祭拜的目的从”求子“转向”溺水”是关键，这里有点生硬。
  5. 赎罪真是永恒的主题。

米泽穗信《满愿》

是短篇小说集，第一次看这个作家，之前知道他是以“青春推理”出名，看完才发现是走黑暗路线的推理小说。

好几篇的心理刻画让我想到乙一，搭配如《石榴》这种略显病态的主题，读完只觉寒意袭人。

最喜欢的是《夜警》，一开始就摆出谜题，然后回忆的部分用很多细节来丰满这个人物，而答案就埋藏在这些细节之中！

秋吉理香子《圣母》

非常精巧的叙述性诡计。

多视角叙事是叙诡的常见结构。其中的两个视角，看上去是完全不相干的人，但其实是生活在同一屋檐下的。这时候如何蒙蔽读者呢？作者设计了另一个小叙诡，很狡猾地掩藏了主角的性别，直到最后才揭示，但这一点很重要，是前面的诡计成立的关键。

这本书前面大部分给人感觉是冰凉不适的，到最后一下子呈现出一种母性光辉般的崇高感。

叶真中显《死亡护理师》

真是非常“硬”的社会派推理小说了，定下一个主题-日本的老龄化问题，一个切入点-老年护理行业，然后开始搭积木一样往里塞人物，代表“善”的检察官，代表“恶”的企业高管，绝望的护理行业底层从业人员，棱角分明，直来直去，中间大段议论文般在探讨日本“老无所依”的困境。

推理只是点缀的小把戏，但也有亮点。一是玩了一个小小的叙述性诡计，多线叙事，其实凶手视角和某个主角视角是重合的；结尾一个小小的反转，给读者的错觉是主角在捕猎“真凶”。另一个有意思的地方，后面破解案件的关键一步是靠着助理检察官基于Excel做的数据分析，虽然用到的统计方法很简单，大概就是基于数据分布发现outlier, 但也算是展现了“小数据”的威力。

从这本书还可以了解点一点日本的司法制度，摘抄一下：

日本的刑事司法号称“精密司法”，判决有罪率99.9%，检察官基于侦查结果做核实，只对确定有罪的案件起诉，所以日本的法庭不是争论有罪无罪的地方，而是执行父爱主义的裁决，检察官向被告问罪，辩护律师陈述被告如何悔过，法官进行教导促其改过。可以看到，日本的法庭审判极具宗教性和道德性。

日本的检察官制度，检察官有侦查权，但很少直接参与案件侦查，大部分工作是于办公室核实案件。

检察官称为“独任官员”，只有检察官有起诉权，哪怕警方逮捕，如果检察官不起诉，也只能无罪释放。

检察官是真正决定一个人有罪与否的人，也可以说是唯一具有合法杀人权力的人，考虑到死刑是最高刑罚。

检察官责任重大，不允许独断专行，要求无条件服从以检察长为首的指挥系统，每个案件都必须向上级汇报，获得批准。

最近读了很多东野圭吾

发表于 2021-06-16 更新于 2021-06-17 分类于 读书

我自己分析主要是两个原因：

一是今年以来工作压力变大，每天上班都感觉有干不完的活，这样下班的时候基本处于身心俱疲的状态。地铁上还是坚持着看书的习惯，但只适合读一些轻松点的休闲小说了。

二是养成了从图书馆借书读的习惯，而去图书馆大部分时候我是从新书区挑书看，结果经常发现怎么东野圭吾又有新书上架了，在纠结于借哪本的时候，东野圭吾就成了比较安全的选择。

也曾试过找比如阿加莎克里斯蒂的书，但家附近的图书馆还是差了一点，竟然找不到阿婆的书，福尔摩斯的书倒有一些。

因为轻松好读，半年以来读过的东野圭吾的书有近十本。真论水准，也是良莠不齐，不过这么高产的情况下有些写烂了也可以理解，阿加莎克里斯蒂也高产，作品中同样也有不少水准之下的。

除了量大更新快，东野圭吾的另一大特点是题材和风格的多样化。以最近看过的几本举例，黎明之街的恋爱心理描写胜过很多爱情小说，超杀人事件和黑笑小说之类专门“编排”推理圈，其中的幽默感也独树一帜，学生街的日子充满青春小说的颓丧感，祈年守护人是和解忧杂货店一样的温情鸡汤路线，我想这也是东野圭吾的一个厉害之处。 很多推理小说家都会有一个特定的标签，比如“本格”，“崩坏”，“暗黑”，同样的风格看几本就会腻。而东野圭吾的标签却很难总结，硬要套一个也只能是“高产”，“流行”，“通俗”之类了。如果碰到那种只能带一个推理作家的书去荒岛的问题，我的答案大概还是东野圭吾。

也正因为太通俗流行（现在进任何一个书店都可以看到至少一大排东野圭吾，文峰的新华书店在畅销区有一个专门的书架全是他的书），完全可以理解某些推理小说迷对他的嫌弃。之前听某播客讲，上海某高校的推理协会在招新时会问你最喜欢的小说家是谁，回答东野圭吾的就会拒掉。

今日的东野圭吾绝对算是功成名就，很多作家在这个地位都会变得爱惜自己的羽毛，惜墨如金。东野圭吾却是反过来越写越多，也许再难有匹及早期嫌疑人和白夜行之类的作品，但这种写作的热情也很令人佩服。
